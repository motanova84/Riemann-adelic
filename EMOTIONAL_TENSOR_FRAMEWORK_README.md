### El Potencial Emocional V(Φ)

```
V(Φ) = (λ/4)(Φ² - Φ₀²)² + μ²Φ² + V_int(Φ,Ψ)
```

**Parámetros:**
- **λ**: Rigidez del sistema (resistencia al cambio emocional)
- **Φ₀**: Estado de "paz fundamental" (mínimo absoluto)
- **μ²**: Masa emocional (inercia afectiva)
- **V_int**: Acoplamiento con la coherencia cuántica Ψ

**Fases del Sistema:**
- `μ² > 0` → Fase restaurada (único mínimo en Φ = 0)
- `μ² < 0` → Ruptura espontánea de simetría (dos mínimos: ±Φ₀)
