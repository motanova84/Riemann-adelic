# Arte Sistema Z — Artefactos Artísticos

**Coherencia Artística**: Ψ = 0.978

Este directorio contiene artefactos artísticos que visualizan los cuatro pilares
del Sistema Dinámico Z para el cierre espectral de la Hipótesis de Riemann.

## Artefactos

1. **demo_1_compactificacion_noconmutativa.png** - Espectro discreto y medida de Haar
2. **demo_2_filtro_adelico.png** - Cancelación de Möbius y función ψ
3. **demo_3_hadamard_determinante.png** - Identidad de Hadamard y fase de Berry
4. **demo_4_dinamico_z.png** - Flujo Anosov y estadísticas GUE
5. **demo_5_sistema_completo.png** - Resumen global con gauge de coherencia
6. **visualizacion_sistema_z.py** - Script ejecutable de visualización

## Ejecutar Visualizaciones

```bash
cd amda/arte_sistema_z
python3 visualizacion_sistema_z.py
```

Generará las 5 visualizaciones principales del sistema.
