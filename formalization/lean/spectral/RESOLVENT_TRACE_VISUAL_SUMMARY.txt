```
┌─────────────────────────────────────────────────────────────────────┐
│                    REPRESENTACIÓN ESPECTRAL                           │

│              [resolvent_spectral theorem]                            │
└────────────────────────────┬────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    CLASE TRAZA DEL RESOLVENTE                         │

│        • Usar criterio de nuclearidad de Grothendieck               │
│        • T ∈ TraceClass ↔ T = A ∘ B (A,B ∈ HilbertSchmidt)         │
└────────────────────────────┬────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    INTERCAMBIO TRAZA-INTEGRAL                         │
│        Tr[∫ f(λ) dE(λ)] = ∫ f(λ) d(Tr∘E)(λ)                        │
│              [trace_integral_swap theorem]                           │
└────────────────────────────┬────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    MEDIDA ESPECTRAL DISCRETA                          │
│        Tr∘E = ∑' δ_{λₙ} (porque espectro es discreto)               │
│          [discrete_spectral_measure theorem]                         │
└────────────────────────────┬────────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    RESULTADO FINAL                                    │
│        Tr[(H_Ψ - z)⁻¹] = ∑' 1/(λₙ - z)                              │
│         [resolvent_trace_expression - MAIN THEOREM]                  │
└─────────────────────────────────────────────────────────────────────┘
```
